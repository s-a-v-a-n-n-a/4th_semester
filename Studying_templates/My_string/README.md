**Мой `std::string`**
=====================
Этот проект - аналогия `std::string`, реализованная для понимания устройства последнего.

**Сборка**
----------
Чтобы собрать проект, выполните следующие команды в командной строке из текущей директории:
<pre><code>
rm build
mkdir build
cd build
cmake ..
make
./main
</code></pre>
