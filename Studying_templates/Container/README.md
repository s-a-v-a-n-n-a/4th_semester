**Мой контейнер**
=================
Этот проект - аналогия `std::vector`, реализованная для понимания устройства последнего.

**Сборка**
----------
Чтобы собрать проект, выполните следующие команды в командной строке из текущей директории:
<pre><code>
mkdir build
cd build
cmake ..
make
./main
</code></pre>
